Ansible role: epel
==========

[![Build Status](https://travis-ci.org/hxpro/ansible-role-epel.svg?branch=master)](https://travis-ci.org/hxpro/ansible-role-epel)

Install EPEL repository for CentOS

Requirements
------------

CentOS, for other distributions has no effect

Example Playbook
----------------

```
    - hosts: servers
      roles:
        - hxpro.epel
```
